## ⚡ The Problem: The High-Stakes Guessing Game
In today's digital education landscape, scaling high-quality teaching is nearly impossible. Companies receive thousands of applications weekly, but determining if someone has the **"Teaching DNA"**—the empathy to handle a frustrated child, the clarity to simplify complex jargon, and the patience to stay calm—usually requires hundreds of expensive human-hours.

**Static resumes don't teach. Humans don't scale.**

## 🚀 The Solution: Value Innovation through Simulation
Cuemath Genesis isn't just an "automation tool"—it's a **Value Innovation** in HR Tech. We have created a high-fidelity, AI-driven simulation environment that moves beyond simple question-and-answer interviews.

Instead of *asking* a tutor if they are patient, we **put them in a room with Alex.**

### Meet Alex.
Alex is our sophisticated AI persona—a curious but easily frustrated 10-year-old student. Alex doesn't just listen; he pushes back. He gets confused. He asks, *"What does that big word mean?"* He says, *"I don't get it, this is too hard!"*

**We evaluate how tutors teach in the heat of the moment.**

---

## 💎 Core Value Propositions

### 1. High-Fidelity Empathy Assessment
Our AI detects tone, simplifies jargon, and measures how a tutor responds to student frustration. It’s the closest thing to a real classroom observation, available 24/7.

### 2. Standardized "Teaching DNA" Scoring
We evaluate every candidate across 5 objective dimensions:
*   **Warmth**: Do they make the student feel safe?
*   **Clarity**: Can they explain the "Why" behind the "What"?
*   **Patience**: How do they react to repetitive confusion?
*   **Simplicity**: Do they avoid technical jargon?
*   **Fluency**: Professional, clear communication.

### 3. Automated Proof-of-Work
During the interview, our AI automatically generates **educational flashcards** based on the tutor's explanations. This serves as an immediate artifact of their teaching quality.

## 🛠️ Tech Stack: Built for Speed & Scale
*   **AI Engine**: Google Gemini 1.5 Flash (Transcription + Chat + Evaluation)
*   **Frontend**: React + Vite + Tailwind CSS (Optimized for low-latency voice)
*   **Backend**: Node.js + Express
*   **Voice**: Browser Web Speech API (Seamless real-time interaction)

## 🏁 Getting Started

### 1. Clone & Install
```bash
# Frontend
cd frontend && npm install

# Backend
cd backend && npm install
```

### 2. Configure Environment
Create a `.env` in the `/backend` folder:
```env
GEMINI_API_KEY=your_key_here
PORT=3001
```

### 3. Launch
```bash
# Terminal 1
cd backend && npm run dev

# Terminal 2
cd frontend && npm run dev
```
